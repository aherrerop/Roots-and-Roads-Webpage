## Important

- Do not overwrite `main_js.js` unless you are applying the tracking snippet manually.
- This package adds crawlable itinerary pages, AI summary files, machine-readable JSON, structured data, external profile references, expanded sitemap, and AI/travel-assistant friendly wording.
- `booking-success/` is optional. It is included for future conversion tracking if you decide to redirect successful bookings there.
- After deployment, open these URLs directly:
  - https://rootsandroadsbcn.com/sitemap.xml
  - https://rootsandroadsbcn.com/robots.txt
  - https://rootsandroadsbcn.com/llms.txt
  - https://rootsandroadsbcn.com/ai-summary.md
