# Roots & Roads AI retrieval implementation

Upload the contents of this ZIP to the GitHub Pages repository root.

Required root files:

- index.html
- robots.txt
- sitemap.xml
- llms.txt
- ai-summary.md
- en/ directory with all generated landing pages

After upload:

1. Open https://rootsandroadsbcn.com/sitemap.xml and confirm all URLs appear.
2. Open https://rootsandroadsbcn.com/llms.txt and https://rootsandroadsbcn.com/ai-summary.md.
3. In Google Search Console, submit sitemap.xml.
4. Use URL Inspection for the homepage and the new itinerary pages.
5. Track impressions for these query families: Barcelona itinerary, what to do in Barcelona, best walking tour Barcelona, Sagrada Família walking tour, German walking tour Barcelona, Spanish walking tour Barcelona.

Main strategic change:

The homepage remains the booking/sales page. The new /en/ pages are retrieval pages matching likely AI and search prompts. They make Roots & Roads easy to insert into itinerary answers as the first-day orientation module.
